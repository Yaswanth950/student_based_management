# Student Data Management System

## Overview
Student Data Management System is a C programming project that allows users to manage student records through a menu-driven interface. Users can add, view, search, update, and delete student details efficiently.

## Features
- Add new student records
- Display all student records
- Search student by Roll Number
- Update student information
- Delete student records
- Count total students
- Exit the application

## Technologies Used
- C Programming
- GCC Compiler
- Command Line Interface (CLI)

## Project Structure

Student-Data-Management/
│── main.c
│── README.md

## How to Run

1. Clone the repository
```
git clone https://github.com/your-username/Student-Data-Management.git
```

2. Navigate to the project folder
```
cd Student-Data-Management
```

3. Compile
```
gcc main.c student.c -o student
```

4. Run
```
./student
```
